# 📋 **박사논문 연구 계획 최종 정리**

---

## 🎯 **1. 연구 개요**

### **연구 주제**
```
노년기 여성의 불안 감소를 위한 온라인 자기자비 글쓰기 중재:
ChatGPT 피드백 효과에 대한 탐색적 혼합방법론 연구
```

### **연구 유형**
- **탐색적(Exploratory) 연구**
- **순차적 혼합방법론** (양적 → 질적)
- 파일럿이 아닌 완결된 연구

### **연구 대상**
- 60-74세 한국인 여성
- 정신질환 진단 이력 없음
- 최종 표본: **30명** (각 집단 10명)

---

## 🔬 **2. 연구 설계**

### **양적 연구: 3집단 준실험설계**

| 집단 | 중재 내용 | 인원 |
|------|----------|------|
| **A집단** (실험군1) | 자기자비 글쓰기 + ChatGPT 실시간 피드백 | 10명 |
| **B집단** (실험군2) | 자기자비 글쓰기 단독 (피드백 없음) | 10명 |
| **C집단** (통제군) | 중립적 글쓰기 (일상 활동 묘사) | 10명 |

### **무선배정 방법**
- **블록 무선배정** (연령대별 블록)
  - 60-64세 / 65-69세 / 70-74세
  - Excel RAND() 함수 또는 봉투 추첨법 사용

## 📊 **3. 측정 도구 (모두 무료!)**

### **사전/사후 측정**

| 도구 | 문항수 | 소요시간 | 측정 내용 |
|------|--------|----------|-----------|
| **SCS-SF-12** | 12 | 3분 | 자기자비 |
| **PANAS-SF-10** | 10 | 2분 | 긍정/부정 정서 |
| **GAS-10** | 10 | 2분 | 노인 불안 ⭐ |

- **GAS-10**: 한국판 노인불안척도 단축형 (무료 사용 가능)
- 인구통계: 연령, 최종학력, 결혼상태, 동거여부, 주요 스트레스 원인

### **중간 측정**
- PANAS-SF-10 (부정적 사건 회상 직후)

### **총 소요 시간: 약 20-25분** ✅

## 🔄 **4. 실험 절차**

```
1단계: 사전검사 (7분)
   ├─ 동의서 및 인구통계
   ├─ SCS-SF-12
   ├─ PANAS-SF-10
   └─ GAS-10

2단계: 부정적 사건 회상 글쓰기 (10분)
   └─ 과거 힘들었던 경험 회상 및 작성

3단계: 중간측정 (2분)
   └─ PANAS-SF-10 (정서 활성화 확인)

4단계: 실험 조작 (10분)
   ├─ A집단: 자기자비 글쓰기 3단계 + GPT 피드백
   │   • 공통인류성 (3분) → 피드백
   │   • 자기친절 (3분) → 피드백
   │   • 마음챙김 (4분) → 피드백
   ├─ B집단: 자기자비 글쓰기 (피드백 없음)
   └─ C집단: 중립적 글쓰기 (일상 활동)

5단계: 사후검사 (10분)
   ├─ SCS-SF-12
   ├─ PANAS-SF-10
   ├─ GAS-10
   └─ ⭐ 서술형 질문 6개 (추가!)
   └─ 추가 인터뷰 참여 의향 조사
```

## 💬 **5. 질적 연구 (핵심 차별화 전략!)**

### **5-1. 서술형 응답 (전체 30명)**

**사후 설문에 포함되는 6개 질문:**

```
Q1. 부정적 경험 회상 및 글쓰기 경험
Q2. [집단별 차별화]
    - A집단: ChatGPT 피드백 경험
    - B집단: 피드백 없는 글쓰기 경험
    - C집단: 중립적 글쓰기 경험
Q3. 불안/스트레스 변화
Q4. 자기 돌봄에 대한 생각 (문화적 맥락)
Q5. 온라인 프로그램 경험
Q6. 일상생활 변화
```

**분석 방법:**
- 귀납적 내용 분석
- 주제 도출 및 빈도 분석
- 집단별 비교

### **5-2. 심층 인터뷰 (4-6명)**

**선정 기준 (Purposive Sampling):**
- 극단적 사례 (큰 변화 또는 무변화)
- 풍부한 서술을 한 사람
- 문화적 갈등 언급자
- 각 집단에서 1-2명

**인터뷰:**
- 온라인 (Zoom/Google Meet)
- 30-40분
- 반구조화 인터뷰
- 녹음 후 전사

**분석 방법:**
- 주제 분석 (Braun & Clarke, 2006)
- 6단계 체계적 분석

## 📈 **6. 자료 분석**

### **양적 분석**

```
도구: SPSS 29.0 (학교 라이선스)

분석 방법:
1. 기술통계 및 정규성 검정
2. 집단 간 동질성 검증
3. 반복측정 혼합설계 ANOVA
   (또는 비모수 검정)
4. ANCOVA (사전점수 공변인)
5. ⭐ 효과 크기 계산 (Cohen's d, partial η²)
6. 95% 신뢰구간 제시

핵심: p-value보다 효과 크기 중심 해석!
```

### **질적 분석**

```
서술형 (30명):
- Excel 정리
- 코딩 및 주제 도출
- 빈도 분석
- 집단별 비교

인터뷰 (4-6명):
- 전사 (Clova Note 무료)
- 주제 분석
- 극단 사례 탐색
- 기제 이해
```

### **통합 분석**

```
양적-질적 통합 매트릭스:
- 삼각검증 (Triangulation)
- 수렴/발산 지점 파악
- 상호 보완적 해석
```

## 💬 **왜 "사후" 라고 했는지**

제가 "사후 설문"이라고 해서 혼란을 드렸습니다. 정확히는:

```
❌ 잘못된 이해:
"실험 다 끝나고 → 나중에 → 별도로 설문"

✅ 올바른 이해:
"실험 직후 → 같은 세션에서 → 바로 이어서 질문"

즉, "post-intervention questionnaire"의 일부로
척도 측정 후 바로 개방형 질문 추가!
```
